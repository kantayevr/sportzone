#  SportZone — Online Fitness Store

# Team Members

##  Project Topic
SportZone — это веб-приложение для покупки спортивных товаров, одежды, инвентаря и добавок.

# Features
- Responsive design (адаптивная вёрстка)
- Light/Dark mode (через localStorage)
- Login/Signup with validation
- Product catalog + Add to Cart
- Search and filter system
- Weather API integration (Open-Meteo)
- Image carousel + animations
